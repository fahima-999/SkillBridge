# SkillBridge

A frontend-only prototype for SkillBridge built with HTML5, CSS3, and Vanilla JavaScript.

## Overview

SkillBridge is a SaaS-inspired career readiness platform that simulates:
- Career gap analysis
- Skill assessment
- Learning roadmap planning
- Resume review
- AI career coaching
- Profile and settings management

## Pages

- `index.html` — Landing page
- `dashboard.html` — User dashboard
- `careers.html` — Career explorer with filtering
- `career-details.html` — Career detail profile
- `assessment.html` — Multi-step skill assessment form
- `results.html` — Skill gap analysis results
- `roadmap.html` — Learning roadmap timeline
- `resume-analysis.html` — Resume evaluation summary
- `ai-coach.html` — AI career coaching chat interface
- `profile.html` — Editable user profile UI
- `settings.html` — Appearance and preferences settings

## Files

- `css/style.css` — Shared global styling and layout
- `css/components.css` — UI components, forms, chat
- `css/dashboard.css` — Dashboard-specific layout
- `css/responsive.css` — Responsive breakpoints
- `js/mock-data.js` — Local mock data objects
- `js/theme.js` — Theme toggle and localStorage persistence
- `js/app.js` — General page interactions and content rendering
- `js/assessment.js` — Multi-step assessment form logic
- `js/dashboard.js` — Dashboard-specific UI setup
- `js/utils.js` — Toast helper utility

## Usage

Open any `.html` file in a browser to view the prototype. The project uses no backend, no frameworks, and no external APIs.

## Notes

- All data is mocked locally using JavaScript objects.
- Theme preferences persist using `localStorage`.
- The assessment form and AI coach are frontend-only demonstrations.
